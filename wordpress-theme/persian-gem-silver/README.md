# قالب وردپرس — Persian Gem Silver

قالب کلاسیک PHP فارسی (RTL) برای فروشگاه جواهرات نقره و سنگ، کاملاً سازگار با **WooCommerce**.

## ✨ ویژگی‌ها
- صفحه اصلی با ۱۹ بخش اختصاصی (Hero، دسته‌بندی‌ها، انگشتر شاخص، پرفروش‌ها، شخصیت‌شناسی سنگ، طراحی اختصاصی، شمش نقره، نظرات، مجله، FAQ، خبرنامه و…)
- صفحه لیست محصولات (PLP) با فیلتر و سورت
- صفحه جزئیات محصول (PDP) با گالری، تب‌ها، Trust Bar و دکمه «طراحی اختصاصی»
- پشتیبانی کامل WooCommerce + کارت محصول سفارشی
- Customizer گسترده برای ویرایش متن‌های هیرو، اعلان، تماس و فعال/غیرفعال کردن هر سکشن
- تایپوگرافی Vazirmatn، طراحی RTL، رنگ‌بندی luxury (سبز emerald + طلایی)
- Responsive کامل + منوی موبایل drawer

## 📦 نصب
1. فایل ZIP قالب را دانلود کنید.
2. در پنل وردپرس بروید به: **نمایش → پوسته‌ها → افزودن → بارگذاری پوسته**.
3. ZIP را آپلود و **فعال** کنید.
4. در فعال‌سازی اول، صفحه‌های پیش‌فرض، منوها و صفحه اصلی به‌صورت خودکار ساخته می‌شوند.

## ⚙️ پیکربندی
- **نمایش → سفارشی‌سازی** را باز کنید:
  - **برند و تماس**: شماره تماس، واتساپ، آدرس، شبکه‌های اجتماعی
  - **نوار اعلان بالایی**: متن باند بالای صفحه
  - **بخش هیرو**: عنوان، متن، دکمه‌ها و تصویر پس‌زمینه
  - **بخش‌های صفحه اصلی**: نمایش/مخفی کردن هر سکشن
- نصب افزونه **WooCommerce** برای تکمیل فروشگاه (PLP، PDP، سبد، پرداخت).
- (اختیاری) **Contact Form 7** برای فرم تماس.

## 🛍 افزودن محصول
- پس از فعال‌سازی WooCommerce، از منوی **محصولات → افزودن محصول** اقدام کنید.
- دسته‌بندی‌های پیشنهادی: انگشتر، دستبند، گردنبند، گوشواره، ست، سنگ، شمش.

## 🎨 سفارشی‌سازی استایل
- متغیرهای رنگ در: `assets/css/theme.css` (CSS variables داخل `:root`)
- استایل صفحه اصلی: `assets/css/home.css`
- استایل PLP: `assets/css/plp.css`
- استایل PDP: `assets/css/pdp.css`

## 📁 ساختار قالب
```
persian-gem-silver/
├── style.css, functions.php
├── header.php, footer.php
├── front-page.php, page.php, single.php, index.php, 404.php
├── inc/  (theme-setup, enqueue, customizer, woocommerce, seed-content, helpers)
├── template-parts/home/  (۱۹ سکشن صفحه اصلی)
├── template-parts/plp/   (کارت محصول)
├── woocommerce/  (archive-product, single-product, content-product)
└── assets/  (css, js, images)
```

## 🚧 نکات
- این قالب React نیست؛ کلاسیک PHP است و نیازی به Node/Build ندارد.
- انیمیشن‌های نسخه React با CSS و IntersectionObserver ساده بازسازی شده‌اند.
- فرم «طراحی اختصاصی» در حال حاضر کلاینت‌ساید است؛ برای ارسال به ایمیل، آن را به CF7 یا WPForms متصل کنید.
